# Gif-Maker-Local
Local gif maker that allows you to use various effects on images turning them into gifs using your local machine to do the work credit to jnordberg https://github.com/jnordberg/gif.js/blob/master/dist/gif.js is integrated to help the process the effects are anime sword slice, shatter explosion, glitch, and wave.
